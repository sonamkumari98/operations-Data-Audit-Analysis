Operations Data Audit & Revenue Analysis
Project Overview
This project focuses on operational precision and data reliability for a Mandir Puja booking system. As a Reliability Architect, I analyzed revenue trends and performed a deep audit of user-submitted data.
Key Tasks
1. Revenue Analysis (Task 1)
Created Pivot Tables to analyze revenue for August, September, and October.
Identified September as the peak month with the highest growth.
2. Data Integrity Audit (Task 2)
Audited 100+ records for Name and Gotra accuracy across English and Hindi scripts.
Categorized 28 critical issues including placeholder entries ("Ram Ram", "Test") and phonetic mismatches.
Tech Stack
Tool: Microsoft Excel (Pivot Tables, Data Validation, Formatting)
Skills: Data Auditing, Operations Management, Multilingual Data Handling.
